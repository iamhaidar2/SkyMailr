Set up DNS for this domain

Mailbox providers use SPF, DKIM, and DMARC to decide whether to trust mail that uses your domain. Without them, messages are more likely to be filtered or rejected.

  1. Open your DNS host (where you bought the domain or your IT team manages DNS).
  2. Add the records below exactly as shown — do not paste example values from other documentation.
  3. Wait for propagation (often minutes, sometimes up to 48 hours).
  4. Return here and use “Check records now” to confirm.

Many DNS panels ask for a short host or name instead of the full hostname. Use the host column when your provider wants only the left-hand label; use full hostnames if your provider stores the complete name.
