How to use your API key

The HTTP API expects Bearer authentication: send header Authorization: Bearer <your-key> on each request (TenantAPIKeyAuthentication).

Put the key in an environment variable (for example SKYMAILR_API_KEY), a secret manager, or other server-only config. Do not expose it in browsers, mobile apps, or version control.

cURL 
curl -sS "{{ api_base_url }}/health/" \
  -H "Authorization: Bearer YOUR_API_KEY"
Python (httpx) 
import os
import httpx

url = "{{ api_base_url }}/health/"
headers = {"Authorization": f"Bearer {os.environ['SKYMAILR_API_KEY']}"}
r = httpx.get(url, headers=headers, timeout=30.0)
r.raise_for_status()
print(r.json())
Node (fetch) 
const url = "{{ api_base_url }}/health/";
const key = process.env.SKYMAILR_API_KEY;
const res = await fetch(url, {
  headers: { Authorization: `Bearer ${key}` },
});
if (!res.ok) throw new Error(await res.text());
console.log(await res.json());
