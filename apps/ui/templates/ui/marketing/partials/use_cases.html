Use cases

SaaS onboarding

Welcome, verify email, and progressive feature unlocks with the same template keys across environments.

Billing & receipts

Invoice and payment notices per tenant with clear sender domains and suppression awareness.

Security & alerts

Password resets, device alerts, and collaborator invites with audit-friendly message IDs.

Lifecycle marketing

Drip sequences and exits when users convert—without bolting on a second ESP for transactional mail.

Marketplaces

Isolate merchants as tenants; operators retain global visibility for abuse and deliverability.

Internal tools

Staff notifications and handoffs with the operator UI for overrides during incidents.
